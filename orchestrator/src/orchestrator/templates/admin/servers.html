Prepare & Snapshot
What does Prepare & Snapshot do?
  1. Detects OS version (vendor, major/minor)
  2. Configures passwordless sudo
  3. Opens firewall port 8080
  4. Installs Java 17 (required for emulator)
  5. Formats and mounts /data disk with output folders
  6. Takes a snapshot — this becomes the OS Level baseline
